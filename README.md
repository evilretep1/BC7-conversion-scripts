Scripts to:

-lowercase all file names in a folder (recursively)
-Generate Atlases for select regions
-convert image files in select formats to .dds with BC7 compression for use in OpenMW 0.51.0 full release

Requirements:
-Linux or WSL
-Texconv (https://github.com/microsoft/DirectXTex)
-ImageMagick (should be pre-installed in your distro, which in my case is CachyOS with the LTS kernel)
